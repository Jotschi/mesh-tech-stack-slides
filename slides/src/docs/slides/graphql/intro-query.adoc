<h2><span class="component">GraphQL</span> - Query</h2>
++++

[source,json]
----
{
  node(path: "/aircrafts/space-shuttle") {
    uuid
    fields {
      ... on vehicle {
        weight
        price
        slug
        description
      }
    }
  }
}
----

++++
    <aside class="notes">
      * Typical Gentics Mesh query
      * We query for weight, price, slug and description
      * Union types can be used to handle dynamic results
    </aside>
